How to install TypeScript?
Answer:
Step #1: Install Node.js package manager
https://nodejs.org/en/
https://nodejs.org/dist/v10.15.0/node-v10.15.0-x64.msi

Step #2: Using command prompt, issue the following command. This will install globally.
> npm install -g typescript

How to check whether TypeScript is installed or not. To check with version?
Answwer: In the command prompt, provide the following command
> tsc --version  or tsc -v 
To find the help
> tsc --help or tsc -h
